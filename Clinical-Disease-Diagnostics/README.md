# Heart Disease Prediction

## Overview

This project predicts whether a patient has heart disease using machine learning classification models. The dataset contains patient health information such as age, cholesterol level, blood pressure, heart rate, and other medical features.

## Dataset

The project uses the Heart Disease dataset with the following features:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise Induced Angina
- Oldpeak
- Slope
- Number of Major Vessels
- Thal
- Target

Target:
- **0** = No Heart Disease
- **1** = Heart Disease

## Machine Learning Models

The following supervised learning models were implemented:

- Support Vector Machine (SVM)
- Random Forest
- XGBoost

## Project Workflow

1. Import required libraries
2. Load the dataset
3. Explore and analyze the data
4. Check missing values
5. Split features and target
6. Split training and testing data
7. Train machine learning models
8. Evaluate model performance
9. Compare model accuracy
10. Predict heart disease for a new patient
