# Predictive Credit Risk Engine

## Objective
Build a machine learning model to predict customer creditworthiness using the German Credit dataset.

## Dataset
- German Credit Dataset
- 1000 records
- 20 input features
- 1 target variable (Creditability)

## Models Used
- Logistic Regression
- Decision Tree

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC

## Results
| Model | Accuracy | Precision | Recall | F1-Score | ROC-AUC |
|-------|---------:|----------:|--------:|---------:|--------:|
| Logistic Regression | 75.0% | 77.85% | 89.13% | 83.11% | 66.34% |
| Decision Tree | 72.5% | 82.68% | 76.09% | 79.25% | 70.30% |

## Conclusion
Logistic Regression provided better overall performance, while Decision Tree achieved higher Precision and ROC-AUC.
